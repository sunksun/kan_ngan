## SQL Script ที่ต้องรันบน Server Database

⭐ แนะนำ: ใช้ไฟล์ update_items_table.sql (ป้องกันการรันซ้ำ)

วิธีที่ 1: ใช้ไฟล์ SQL Script (แนะนำ)
-------------------------------------------
1. เข้า phpMyAdmin บน Server
2. เลือก Database: sunksund_kan_ngan
3. คลิก Import > Choose File
4. เลือกไฟล์: update_items_table.sql
5. คลิก Go

หรือ MySQL CLI:
mysql -u sunksund_kan_ngan -p sunksund_kan_ngan < update_items_table.sql

วิธีที่ 2: Copy-Paste SQL (สำหรับครั้งแรกเท่านั้น)
------------------------------------------------------
เข้า phpMyAdmin > SQL tab แล้ว Paste คำสั่งนี้:

```sql
USE sunksund_kan_ngan;

-- ตรวจสอบว่าคอลัมน์มีอยู่แล้วหรือยัง (ป้องกัน error)
ALTER TABLE items
ADD COLUMN IF NOT EXISTS middle_price DECIMAL(10,2) AFTER item_name,
ADD COLUMN IF NOT EXISTS middle_unit VARCHAR(50) AFTER middle_price;

-- ตรวจสอบโครงสร้างตาราง
DESCRIBE items;

-- ตรวจสอบว่ามีข้อมูลในตาราง middle_price
SELECT COUNT(*) as total FROM middle_price;
```

⚠️ หมายเหตุ:
- Script ป้องกันการรันซ้ำ (ถ้ามีคอลัมน์แล้วจะไม่ error)
- ใช้เวลาประมาณ 1-2 วินาที
- ไม่กระทบข้อมูลเดิม

## Troubleshooting

ปัญหา: Autocomplete ไม่ทำงาน
✅ ตรวจสอบ search_middle_price.php permissions = 644
✅ ตรวจสอบ Console ใน Browser (F12) หา error
✅ ทดสอบ API โดยตรงที่ URL ข้างบน

ปัญหา: Database connection error
✅ ตรวจสอบ connect_db.php มี credentials ถูกต้อง
✅ ตรวจสอบว่ารัน SQL ALTER TABLE แล้ว

ปัญหา: PDF ไม่แสดงราคากลาง
✅ ตรวจสอบว่าข้อมูลในตาราง items มี middle_price และ middle_unit
✅ ทดสอบบันทึกข้อมูลใหม่ที่ forms-items.php
